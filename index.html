Level3CacheClear

Cache clearing script for the Level 3 REST API, written in ruby.

Download this project as a .zip file Download this project as a tar.gz file

What is Level3CacheClear?

Level3CacheClear is a script that makes it easy to invalidate cached HTTP objects that are served using the CDN service from Level 3. The script can be called from a command line, or easily called as part of a job.

Getting it working.

...Is simple.

  1. Download Download and extract the package.
  2. Make sure you have execute permissions on cacheclear
  3. Make sure the appropriate gems are installed.
    openssl base64 httparty builder optparse
  4. Update the secretfile with your Level 3 API information.

Thats it! Now just run the script,

./cacheclear -p /your/path/to/clear -h hostname
